1. Project Topic:
Cloudly — an online perfume shop where users can browse, explore, and purchase various fragrances.

2. Overview
This project is a multi-page web application designed to present a modern perfume store experience.
It includes dynamic product pages, a user profile and authorization system, and aesthetic visuals powered by the Pexels API.

3. Technologies Used

Frontend: HTML, CSS, JavaScript

API: Pexels API
 — for high-quality images and visual content

4. Features

1) View perfume brands and product catalog

2) Explore detailed product pages

3) User registration, login, and profile management

4) Contact page and welcoming start screen

5) Beautiful images integrated via Pexels API
